# 💳 ATM Simulation (JavaScript)

This is a simple ATM simulation project built using JavaScript.  
It demonstrates basic programming concepts like switch-case, variables, and conditional statements.

## 🚀 Features
- Check account balance
- Deposit money
- Withdraw money
- Exit option

## 🛠️ Technologies Used
- HTML
- JavaScript

## 📌 How It Works
- The program uses a predefined ATM option.
- Based on the selected option:
  - 1 → Check Balance
  - 2 → Deposit Money
  - 3 → Withdraw Money
  - 4 → Exit
- It performs operations and displays the result on the browser.

## ▶️ How to Run
1. Download or clone the repository
2. Open the HTML file in any browser
3. Output will be displayed on the screen

## 📚 Learning Outcome
- Understanding of switch-case statements
- Basic financial logic implementation
- JavaScript fundamentals
